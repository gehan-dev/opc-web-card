# OPC 网络名片 — 上海专场 Skill

帮助你从零开始搭建「网络名片」并部署到阿里云 ECS。

## 什么是网络名片？

一个官网、一个落地页、一个作品集 — 都是你的网络名片。云服务器负责稳定承载这些内容，让别人能随时访问你的作品。

## 使用方式

### Qoder 用户

在 Qoder 中添加 Skill，输入本仓库地址：

```
https://github.com/gehan-dev/opc-web-card
```

然后对 AI 说「我想搭建我的网络名片」即可开始。

### 其他 AI 工具用户（ChatGPT / Claude / 通义 / Kimi 等）

将以下内容发给你的 AI 助手：

```
请阅读以下 GitHub 仓库的内容，然后一步步引导我完成「网络名片」的搭建和部署：
https://github.com/gehan-dev/opc-web-card

从 SKILL.md 开始，按 Phase 0 → Phase 7 的顺序引导我。
```

## 前提条件

- 阿里云账号（已实名认证）
- 账户余额 ≥ 100 元
- 一个 AI 助手（Qoder / ChatGPT / Claude / 通义 / Kimi 等）
